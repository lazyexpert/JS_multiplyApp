Options page help:

The format "+15/-1 sec" means that you recieve + 15 seconds for each correct answer, and loose -1 second for each wrong one.

Operations type, i believe, its pretty clear. Just basic math operations.

Maximum number value - is a number, within which the tasks will be generated.

Only multiply table. This options allows you to generate tasks out from multiply table only. However, you will be albe to pick any math operations you wish, but multiply and divide will be generated in special way to fit multiply table.

Master Tips

To increase your speed try using only keyboard. Its rather convinient to do on the keynums, where second enter is placed.

Use enter to switch to the next question.

Remeber the time you spend thinking, wont come back, especially if you answer wrong. There might be situations, when you should just pass to another question.
